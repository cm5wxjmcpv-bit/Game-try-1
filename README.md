# Game-try-1

## Tile Texture Layer (Optional)
- Put texture images in `textures/`:
  - `textures/floor1.png`
  - `textures/floor2.png`
  - `textures/floor3.png`
  - `textures/water.png`
  - `textures/lava.png`
- Edit the `TILE_TEXTURE_PACKS.default` paths/fallback colors in `index.html` to change what texture each tile uses.
- If an image is missing or fails to load, the game automatically uses the fallback color and keeps running.
